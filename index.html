Function Documentation

capitalize(str, opts)

Capitalizes the first letter of a string.

capitalize('hello') → 'Hello'

capitalize.words(str, opts)

Capitalizes each word in a string with advanced options.

capitalize.words('hello world') → 'Hello World'

capitalizeWords(str)

Simple capitalization for words split by space.

capitalizeWords('hello world') → 'Hello World'

truncate(str, maxLength, suffix)

Truncates a string and appends a suffix if it exceeds maxLength.

truncate('Hello world', 8) → 'Hello...'

slugify(str)

Converts a string to a URL-friendly slug.

slugify('Hello World!') → 'hello-world'
